# Apple-Picker
Project 1 apple picker repository
Made by Ethan Lemieux for Game Developement

HOW TO PLAY:
Launch the applepicker.exe to play, use the buttons to start the game
Use the arrow keys or 'A' or 'D' to move left and right
Catch apples and avoid rocks
if you miss an apple you lose a life.

ADDITIONAL FEATURES:
There are Golden Apples that give out 10 pts each
every 50 pts, a round increases
Everytime a round increases movement speed and drop rate increases
